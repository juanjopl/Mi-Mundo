Comudin

PHP JavaScript SCSS

El objetivo de este proyecto es desarrollar una plataforma web que combina la experiencia visual, característica de aplicaciones como Instagram, con la comunicación rápida y dinámica que ofrece X. Además, la plataforma cuenta con dos secciones diferenciadas: una destinada a publicaciones de un tipo específico y otra para publicaciones de distinta naturaleza. Los usuarios pueden interactuar no solo a través de comentarios, sino también mediante un sistema de mensajería directa, fomentando una comunicación más completa y eficiente.

Shopee

PHP JavaScript

El objetivo de este proyecto consiste en una plataforma web similar a Wallapop. En ella, los usuarios pueden publicar los productos que desean vender y realizar ofertas por aquellos disponibles. Además, en el proceso de negociación, comprador y vendedor tienen la posibilidad de intercambiar una serie de contraofertas hasta alcanzar un acuerdo. Este desarrollo se llevó a cabo como parte del aprendizaje de PHP en el curso de DAW, lo que me permitió consolidar mis conocimientos sobre la lógica de negocio en aplicaciones web dinámicas.

WikiRicky

JavaScript React

Este proyecto consiste en una aplicación web interactiva que permite explorar a los personajes de la serie Rick & Morty. Con un diseño intuitivo y responsive, los usuarios pueden navegar por una galería dinámica donde se muestra información clave de cada personaje, como su nombre, especie y ubicación, obtenida directamente de la API oficial de Rick & Morty.

Cajero Automático

Java

En este proyecto he recreado un cajero automático con una interfaz gráfica intuitiva que permite al usuario realizar operaciones bancarias comunes: retirar dinero, consultar saldo, realizar depósitos y cambiar el PIN. El desarrollo aplica principios de la Programación Orientada a Objetos (POO), manejo de excepciones y validaciones para garantizar la integridad de los datos y una experiencia de usuario fluida.
